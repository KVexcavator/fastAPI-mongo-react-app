1. там же, в search models окне, выбираем llama 3.1.
выбираем 8b(или больше если пк тянет)
копируем, что то вроде 
ollama run llama3.1 
втравляем в терминал и запускаем
--------------------------------------------------
для запуска модели
ollama run llama3.2
кастомизировал
ollama run excavator
--------------------------------------------------
для работы в терминале команды вводятся через символ слеш /
/bye - окончить сеанс
/help - справка
/serve - запускает локальный сервер для работы с моделями через апи
/create -создаёт новую модель из Modelfile
/show - показвает информацию о выбранной модели
/run - запускает модель для выполнения задач
/pull -загручает модель из реестра
/push-отправляет модель в реестр
/list - отображает список всех доступных моделей
/cp-копирует модель
/rm  - удалить модель с пк
